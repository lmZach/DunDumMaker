📖 Game Rules & Card Types Reference

🎯 Game Goal

  • Party Wins: Collectively reach the required Glory target
  • Chaos Master Wins: If Party fails after 5 Short Rests
  • Loser: Becomes Chaos Master next game

🃏 Card Types Explained

📘 Party Side Cards

Classes - Your character's foundation
  • Dealt once at setup (permanent)
  • Provides base dice and passive ability
  • Draw your character on the dry-erase card!
  • Example: Monk - Way of the Couch Potato (+2 when sitting/lying down, D6)
Subclasses - Your character's special twist
  • Drawn at first Short Rest only (1 per player)
  • Combine with Class to create your Signature Move
  • Use Signature Move once per Short Rest
  • Example: Path of Dramatic Sighing + Monk = "Sigh of Ultimate Laziness"
Stuff & Thangs - Your actions, gear, and spells
  • Mixed deck: Gear, Spells, Actions
  • Start with 5 cards, draw back to 5 at each Short Rest
  • Play 1 main + 1 optional bonus each round
  • Example: Pool Noodle of +1 Rage (+3 if you yell "KABOOM!")

👹 Chaos Master Cards

Creeps - Monsters that compete
  • Chaos Master plays before flipping Dungeon
  • Monster rolls against the Party
  • If monster wins: no Glory + funny penalty
  • Example: Karen the Karen Dragon (d8+2, demands manager)
Curses - Instant chaos effects
  • Played by Chaos Master at round start
  • Immediate effect on Party or specific players
  • No dice roll, just suffer
  • Example: Kevin's Veto (force one player to reroll)
Casualties - Random wild events
  • Roll-table style effects
  • Can help OR hurt the Party
  • Usually involves a d6 or d20 roll
  • Example: Wild Table Magic (Roll d6: 1=lose Glory, 6=free Short Rest)

🏰 Dungeon Cards

Dungeons - The challenges you face
  • Flipped by Chaos Master each round
  • Four subtypes determine how Glory is won:
TypeHow to Win
🗡️ CombatHighest roll wins 1 Glory
🗣️ SocialParty votes for winner (give 1 Glory)
🧭 ExplorationClosest to target number wins 1 Glory
❓ WildChaos Master decides winner

🔄 Round Flow

  1. Chaos Master Phase: Play 1 card (Creep/Curse/Casualty) + flip Dungeon
  2. Party Action: Each player plays 1 main + 1 optional bonus card (face-down)
  3. Roll Phase: Everyone rolls (d20 + bonuses). Creeps roll if active.
  4. Resolution: Dungeon type determines winner & Glory

🛏️ Short Rest (Every 2 Rounds)

  1. Someone rolls d20
  2. Perform the Rest Event for that number
  3. All players draw back to 5 Stuff/Thangs
  4. First Short Rest only: Draw 1 Subclass & create Signature Move

Game ends after 5 Short Rests if Party hasn't won yet!

🎭 Flavor Rules

  • Draw your character silly on the dry-erase card
  • Yell, complain, do voices!
  • Have fun with Rest Events
  • Remember: It's called Dumbassery for a reason 😄

📚 Google Sheets API Setup Instructions

To connect this tool to your Google Sheets, follow these steps:

  1. Enable Google Sheets API:
    • Go to Google Cloud Console
    • Create a new project (or select existing)
    • Enable "Google Sheets API" in the library
  2. Create API Credentials:
    • Go to "Credentials" → "Create Credentials" → "API Key"
    • Copy your API key
    • Paste it in the field below
  3. Get Your Spreadsheet ID:
    • Open your Google Sheet
    • Look at the URL: https://docs.google.com/spreadsheets/d/YOUR_SPREADSHEET_ID/edit
    • Copy the YOUR_SPREADSHEET_ID part
    • Paste it below
  4. Share Your Sheet:
    • Click "Share" in your Google Sheet
    • Change to "Anyone with the link can view"
    • Save
